Лекція 1. Корисні матеріали.

  1. Современный учебник JavaScript

    2. Справочники:
  2. Mozilla Developer Network
    Содержит информацию, верную для основных браузеров. Также там присутствуют расширения только для Firefox (они помечены).
  3. MSDN
    Cправочник от Microsoft. Там много информации, в том числе и по JavaScript (они называют его «JScript»). Если нужно что-то, специфичное для IE — лучше лезть сразу туда.
  4. Safari Developer Library
  5. Dottoro
    Подробная информация по HTML/CSS/JavaScript с учётом браузерной совместимости.
  6. JavaScript.ru
    Cправочник по JavaScript на русском языке.

    7. Редакторы кода (IDE):
  7. WebStorm
    Интегрированная среда разработки на JavaScript, CSS & HTML от компании JetBrains, разработанная на основе платформы IntelliJ IDEA. WebStorm обеспечивает автодополнение, анализ кода на лету, навигацию по коду, рефакторинг, отладку, и интеграцию с системами управления версиями
  8. Sublime Text
    Кроссплатформенный проприетарный текстовой редактор. Поддерживает плагины на языке программирования Python. Инструментарий программы удобен для редактирования исходных текстов программ. Официальный сайт позволяет бесплатно и без ограничений ознакомиться с продуктом, однако уведомляет об обязанности приобретения лицензии.
  9. Atom
    Бесплатный текстовый редактор с открытым исходным кодом для OS X, Linux, Windows. C поддержкой плагинов, написанных на Node.js, и встраиваемых под Git управлением. Большинство проходящих пакетов имеют свободные лицензии на программное обеспечение, сообщества строится и поддерживается.

    10. Инструменты разработчика. Отладка и запуск кода.
  10. Google Chrome
    Браузер содержит встроенные средства отладки JS-кода. Один из самых современных и новых браузеров в плане поддержки стандартов.
  11. Firefox с расширением Firebug
    Еще один популярный инструмент разработки.

    12. Руководства по стилю кода та автоматизовані засоби перевірки коду:
  12. Google JavaScript Style Guide
  13. JQuery Core Style Guidelines
  14. JavaScript.ru
  15. Closure Linter
    Проверка на соответствие Google JavaScript Style Guide.
  16. JSLint
    Проверяет код на соответствие стилю JSLint, в онлайн-интерфейсе вверху можно ввести код, а внизу различные настройки проверки, чтобы сделать её более мягкой.
